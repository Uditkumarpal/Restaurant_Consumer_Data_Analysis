## 🎯 About the Project

This project delivers an end-to-end **Restaurant Consumer Analytics Dashboard** built with **Microsoft Power BI**, analyzing real-world restaurant and consumer data from **3 Mexican states** — San Luis Potosi, Morelos, and Tamaulipas — across **4 cities** including San Luis Potosi, Ciudad Victoria, Cuernavaca, and Jiutepec.

The dashboard is structured into **5 analytical pages** — Home Overview, Dining Preferences, Hospitality Analysis, Consumer Behaviour, and Restaurant Reviews — delivering a complete 360° view of the restaurant ecosystem and consumer patterns.

This project demonstrates complete **Business Intelligence** expertise including data modeling across 5 related tables, advanced DAX measure creation, Power Query data transformation, cross-filter interactions, and professional dashboard design using a consistent dark and neutral color theme.
